## IPTV

自动收集整理，优先使用高频的直播源

### 使用

#### 直接调用

每天自动生成

```txt
https://raw.githubusercontent.com/JinnLynn/iptv/dist/live.m3u
```

```txt
https://raw.githubusercontent.com/JinnLynn/iptv/dist/live.m3u
```

#### 手动生成

```shell
pip install -r requirements.txt
python iptv.py
```
